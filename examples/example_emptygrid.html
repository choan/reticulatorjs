Use an empty grid adding custom guides

You only need to include the reticulator.js file and make a call like this

  
    var grid = new Reticulator({
      columns: 0,
      opacity: 0.8
    });
    
    grid.addHorizontalGuide(70);
    grid.addHorizontalGuide(127);
    
    grid.addVerticalGuide(710);
    grid.addVerticalGuide(719);
